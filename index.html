Molang Documentation

Introduction

Welcome to the Molang Editor. This tool allows you to create and test Molang expressions for use in Minecraft's animation and rendering systems.

Basic Usage

  • Write your Molang expressions in the editor
  • Click the Run button to execute your code
  • Use the Copy Molang button to copy your code
  • Load examples from the dropdown menu

Common Functions

query.variable('var_name')

Access a variable value

math.sin(time * 0.1)

Use mathematical functions

variable.my_var = 1.0;

Set variable values

Tips

  • Use the console for debugging output
  • Variables must be declared before use
  • Check the browser console for error messages
  • Press Ctrl+Space for code completion
